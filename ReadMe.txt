############################# APPLICATION DE LISTE DE COURSE #####################################################################

--->Description : gerez votre liste de course sans oublier une seule

			Avec les fonctionnalités de : --> ajouter un article
								--> supprimer un article
								--> marquer un article comme acheté
								--> réorganiser la liste selon vos priorité (du haut vers le bas)

---> Interface :
		--> champ d'ajout: -->  entre l'article que vous souhaitez (longueur maximale : 30 caractères)
							
					--> boutton plus : clicquez sur le "boutton plus" pour valider
		
		--> affichage :  --> les articles sont affichés par ordre d'ajout du haut vers le bas
						
					--> pour réorganiser la liste, faite un "glissez-déposer" de l'article vers le bas
		
					--> pour marquer un article "acheté": coché la case à gauche

					--> pour supprimer un article: cliquez sur le "boutton delete" en rouge
				
					--> pour vider toute la liste : cliquez sur le boutton "vider la liste"

NB: pour une usage optimal:
1- enregistrez tous vos articles 
2- reorganisez l'ordre selon votre priorité
3- marquez ensuite les articles achetés au fur et à mesure

Bon usage ;-)
